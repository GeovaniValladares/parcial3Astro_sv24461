Este documento sirve como el **Manual del Programador** para la aplicación desarrollada en el marco del Parcial III de la materia Herramientas de Inteligencia Artificial para el Desarrollo Web.

### Objetivo del Proyecto
Desarrollar una aplicación web full-stack utilizando **Astro**, integrando un sistema de autenticación (Lucía Auth), una base de datos relacional (Astro DB) y un panel administrativo con operaciones CRUD y paginación.

### Tecnologías Utilizadas
- **Framework:** Astro 6
- **Autenticación:** Lucía Auth
- **Base de Datos:** Astro DB (libSQL)
- **Estilos:** Tailwind CSS
- **Componentes Interactivos:** SweetAlert2
- **Documentación:** Starlight
